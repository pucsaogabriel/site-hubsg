# Site PucTec São Gabriel
Este repositório contém o desenvolvimento de um site voltado à divulgação e gestão de um grande projeto de pesquisa universitária com viés empreendedor. A plataforma tem como objetivo centralizar informações relevantes, compartilhar avanços científicos, conectar pesquisadores e facilitar a comunicação com a comunidade acadêmica e o público externo.

O projeto integra conceitos de pesquisa, inovação e tecnologia, servindo como ponte entre o ambiente acadêmico e o ecossistema de empreendedorismo, promovendo colaboração, transparência e impacto social.
